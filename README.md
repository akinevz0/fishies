# fishies
A game about being a fish

## what this game is about

*you are a fish* - you know how to swim, and how to eat **stuff**

##### Stuff to eat 

- other fish
- eels
- seaweed
- crabs
- moluscs

The bigger you are, the bigger your food is. 

Become bigger than the whales.
